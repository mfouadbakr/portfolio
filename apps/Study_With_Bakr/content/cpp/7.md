# 7. Introduction to OOP in C++

Object-Oriented Programming (OOP) is a programming paradigm based on objects and classes.

---

## What is a Class?

A class is a blueprint for creating objects.

    class Person {
    public:
        string name;
        int age;
    };

---

## Creating an Object

    Person p1;

    p1.name = "Ali";
    p1.age = 20;

---

## Member Functions

Functions inside a class.

    class Person {
    public:
        string name;

        void greet() {
            cout << "Hello, " << name;
        }
    };

    int main() {
        Person p;
        p.name = "Ali";
        p.greet();
    }

---

## Access Specifiers

- `public` → accessible from outside  
- `private` → accessible only inside class  
- `protected` → used in inheritance  

---

## Example with Private Members

    class Person {
    private:
        int age;

    public:
        void setAge(int a) {
            age = a;
        }

        int getAge() {
            return age;
        }
    };

---

## Constructor

Special function called when object is created.

    class Person {
    public:
        string name;

        Person(string n) {
            name = n;
        }
    };

    int main() {
        Person p("Ali");
        cout << p.name;
    }

---

## Default Constructor

    class Person {
    public:
        Person() {
            cout << "Object created";
        }
    };

---

## Destructor

Called when object is destroyed.

    class Person {
    public:
        ~Person() {
            cout << "Object destroyed";
        }
    };

---

## This Pointer

Refers to current object.

    class Person {
    public:
        string name;

        void setName(string name) {
            this->name = name;
        }
    };

---

## Summary
- Class is a blueprint
- Object is an instance
- Use access specifiers to control access
- Constructors initialize objects
- Destructor cleans up
