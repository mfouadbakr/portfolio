# 1. Introduction to C++

## What is C++?
C++ is a powerful, high-performance programming language developed by Bjarne Stroustrup.  
It is widely used for system programming, game development, embedded systems, and competitive programming.

---

## Basic Structure of a C++ Program

    #include <iostream>
    using namespace std;

    int main() {
        cout << "Hello, World!";
        return 0;
    }

### Explanation:
- `#include <iostream>` → Library for input/output
- `using namespace std;` → Avoids writing `std::` repeatedly
- `main()` → Entry point of the program
- `cout` → Output stream
- `return 0;` → Ends program successfully

## Input and Output

    #include <iostream>
    using namespace std;

    int main() {
        int age;
        cout << "Enter your age: ";
        cin >> age;
        cout << "You are " << age << " years old.";
        return 0;
    }

## Operators

### Arithmetic Operators:
- `+` Addition
- `-` Subtraction
- `*` Multiplication
- `/` Division
- `%` Modulus

### Example:

    int a = 10, b = 3;
    cout << a + b; // 13
    cout << a % b; // 1

## Simple Example Program
