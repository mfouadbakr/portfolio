# 2. Control Flow in C++

Control flow statements allow you to control the execution of your program based on conditions and loops.

---

## If Statement

Used to execute a block of code if a condition is true.

    int age = 18;

    if (age >= 18) {
        cout << "You are an adult.";
    }

---

## If-Else Statement

Executes one block if true, another if false.

    int number = 5;

    if (number % 2 == 0) {
        cout << "Even";
    } else {
        cout << "Odd";
    }

---

## Else If Ladder

Used when multiple conditions are needed.

    int marks = 85;

    if (marks >= 90) {
        cout << "Grade A";
    } else if (marks >= 75) {
        cout << "Grade B";
    } else if (marks >= 50) {
        cout << "Grade C";
    } else {
        cout << "Fail";
    }

---

## Nested If

An if statement inside another if.

    int age = 20;
    bool hasID = true;

    if (age >= 18) {
        if (hasID) {
            cout << "Allowed entry";
        }
    }

---

## Switch Statement

Used for multiple fixed choices.

    int day = 3;

    switch (day) {
        case 1:
            cout << "Monday";
            break;
        case 2:
            cout << "Tuesday";
            break;
        case 3:
            cout << "Wednesday";
            break;
        default:
            cout << "Invalid day";
    }

---

## While Loop

Repeats while condition is true.

    int i = 1;

    while (i <= 5) {
        cout << i << endl;
        i++;
    }

---

## Do-While Loop

Executes at least once, then checks condition.

    int i = 1;

    do {
        cout << i << endl;
        i++;
    } while (i <= 5);

---

## For Loop

Best when number of iterations is known.

    for (int i = 1; i <= 5; i++) {
        cout << i << endl;
    }

---

## Break and Continue

### Break → exits loop
### Continue → skips current iteration

    for (int i = 1; i <= 5; i++) {
        if (i == 3) continue;
        if (i == 5) break;
        cout << i << endl;
    }

---

## Summary
- Use `if` for decisions
- Use `switch` for fixed options
- Use loops (`for`, `while`, `do-while`) for repetition
- Control loops using `break` and `continue`
