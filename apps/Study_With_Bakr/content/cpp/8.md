# 8. Inheritance & Polymorphism in C++

Inheritance allows a class to inherit properties and methods from another class.  
Polymorphism allows functions or methods to behave differently based on context.

---

## Inheritance

### What is Inheritance?

It allows one class (child) to use properties of another class (parent).

---

## Example of Inheritance

    class Animal {
    public:
        void speak() {
            cout << "Animal makes a sound";
        }
    };

    class Dog : public Animal {
    public:
        void bark() {
            cout << "Dog barks";
        }
    };

    int main() {
        Dog d;
        d.speak(); // inherited
        d.bark();  // own method
    }

---

## Types of Inheritance

- Single Inheritance
- Multiple Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance

## Function Overriding

Redefining a base class function in the derived class.

    class Animal {
    public:
        void speak() {
            cout << "Animal sound";
        }
    };

    class Dog : public Animal {
    public:
        void speak() {
            cout << "Dog barks";
        }
    };

---

## Polymorphism

Means "many forms".

---

## Compile-Time Polymorphism

### Function Overloading

    class Math {
    public:
        int add(int a, int b) {
            return a + b;
        }

        double add(double a, double b) {
            return a + b;
        }
    };

---

## Runtime Polymorphism

### Using Virtual Functions

    class Animal {
    public:
        virtual void speak() {
            cout << "Animal sound";
        }
    };

    class Dog : public Animal {
    public:
        void speak() {
            cout << "Dog barks";
        }
    };

    int main() {
        Animal* a;
        Dog d;
        a = &d;

        a->speak(); // calls Dog's version
    }

---

## Base Class Pointer

    Animal* ptr;
    Dog d;
    ptr = &d;

---

## Summary
- Inheritance promotes code reuse
- Child class inherits from parent
- Overriding changes behavior
- Polymorphism allows flexibility
- Virtual functions enable runtime polymorphism
