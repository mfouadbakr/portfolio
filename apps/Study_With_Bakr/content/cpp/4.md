# 4. Arrays & Concatenation in C++

Arrays allow you to store multiple values of the same type in a single variable.  
Concatenation is mainly used with strings to combine values.

---

## What is an Array?

An array stores multiple elements in a single variable.

### Example:

    int numbers[5] = {1, 2, 3, 4, 5};

---

## Accessing Array Elements

    int numbers[3] = {10, 20, 30};

    cout << numbers[0]; // 10
    cout << numbers[1]; // 20
    cout << numbers[2]; // 30

---

## Modifying Array Elements

    int numbers[3] = {10, 20, 30};

    numbers[1] = 50;

    cout << numbers[1]; // 50

---

## Looping Through an Array

### Using for loop:

    int numbers[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        cout << numbers[i] << endl;
    }

---

## Array Size

    int numbers[] = {1, 2, 3, 4, 5};

    int size = sizeof(numbers) / sizeof(numbers[0]);

---

## Multidimensional Arrays

### 2D Array Example:

    int matrix[2][2] = {
        {1, 2},
        {3, 4}
    };

    cout << matrix[0][1]; // 2

---

## Looping Through 2D Array

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

---

## Strings in C++

C++ provides a built-in string type.

    #include <string>

    string firstName = "Ali";
    string lastName = "Hassan";

---

## String Concatenation

### Using + operator:

    string fullName = firstName + " " + lastName;

    cout << fullName;

---

### Using append():

    string fullName = firstName;
    fullName.append(" ").append(lastName);

---

## String Length

    cout << fullName.length();

---

## Accessing Characters

    string name = "Ali";

    cout << name[0]; // A
    cout << name[1]; // l

---

## Summary
- Arrays store multiple values
- Index starts from 0
- Use loops to traverse arrays
- Strings can be concatenated using `+` or `append()`
