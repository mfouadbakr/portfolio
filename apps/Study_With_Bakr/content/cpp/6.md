# 6. Structures (Struct) in C++

Structures (structs) allow you to group different data types under one name.

---

## What is a Struct?

A struct is a user-defined data type that groups related variables.

    struct Person {
        string name;
        int age;
        float height;
    };

---

## Creating Variables of Struct

    Person p1;

    p1.name = "Ali";
    p1.age = 20;
    p1.height = 1.75;

---

## Accessing Struct Members

    cout << p1.name << endl;
    cout << p1.age << endl;
    cout << p1.height << endl;

---

## Struct Initialization

    Person p2 = {"Omar", 25, 1.80};

---

## Array of Structs

    Person people[2] = {
        {"Ali", 20, 1.75},
        {"Sara", 22, 1.65}
    };

---

## Loop Through Struct Array

    for (int i = 0; i < 2; i++) {
        cout << people[i].name << endl;
    }

---

## Nested Struct

    struct Address {
        string city;
        string country;
    };

    struct Person {
        string name;
        Address addr;
    };

    Person p;
    p.name = "Ali";
    p.addr.city = "Cairo";
    p.addr.country = "Egypt";

---

## Pointer to Struct

    Person p = {"Ali", 20, 1.75};
    Person* ptr = &p;

    cout << ptr->name;
    cout << ptr->age;

---

## Struct vs Class

| Struct | Class |
|-------|------|
| Default access: public | Default access: private |
| Used for simple data | Used for OOP |

---

## Summary
- Struct groups related variables
- Use `.` to access members
- Use `->` with pointers
- Can create arrays and nested structs
