# 3. Functions & Recursion in C++

Functions help organize code into reusable blocks. Recursion is when a function calls itself.

---

## What is a Function?

A function is a block of code that performs a specific task.

### Example:

    #include <iostream>
    using namespace std;

    void greet() {
        cout << "Hello!";
    }

    int main() {
        greet();
        return 0;
    }

---

## Function with Parameters

    void greetUser(string name) {
        cout << "Hello, " << name;
    }

    int main() {
        greetUser("Ali");
    }

---

## Function with Return Value

    int add(int a, int b) {
        return a + b;
    }

    int main() {
        int result = add(3, 4);
        cout << result;
    }

---

## Default Parameters

    void greet(string name = "Guest") {
        cout << "Hello, " << name;
    }

    int main() {
        greet();        // Guest
        greet("Omar");  // Omar
    }

---

## Function Overloading

Same function name, different parameters.

    int add(int a, int b) {
        return a + b;
    }

    double add(double a, double b) {
        return a + b;
    }

---

## Recursion

A function calling itself.

### Important:
- Must have a **base case**
- Prevents infinite loop

---

## Example: Factorial

    int factorial(int n) {
        if (n == 0)
            return 1;
        return n * factorial(n - 1);
    }

    int main() {
        cout << factorial(5); // 120
    }

---

## Example: Fibonacci

    int fib(int n) {
        if (n <= 1)
            return n;
        return fib(n - 1) + fib(n - 2);
    }

    int main() {
        cout << fib(6); // 8
    }

---

## Recursion vs Iteration

| Recursion | Iteration |
|----------|----------|
| Uses function calls | Uses loops |
| Cleaner code | Faster performance |
| Uses more memory | Memory efficient |

---

## Summary
- Functions improve code organization
- Parameters allow input
- Return values give output
- Recursion solves problems by self-calling
