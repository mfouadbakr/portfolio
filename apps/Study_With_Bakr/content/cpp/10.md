# 10. File Handling in C++

File handling allows you to store data permanently in files and retrieve it later.

C++ provides file handling through the `<fstream>` library.

---

## File Streams in C++

- `ofstream` → Write to file  
- `ifstream` → Read from file  
- `fstream` → Both read and write  

---

## Writing to a File

    #include <fstream>
    using namespace std;

    int main() {
        ofstream file("data.txt");

        file << "Hello, World!" << endl;
        file << "C++ File Handling";

        file.close();
        return 0;
    }

---

## Reading from a File

    #include <fstream>
    #include <iostream>
    using namespace std;

    int main() {
        ifstream file("data.txt");
        string line;

        while (getline(file, line)) {
            cout << line << endl;
        }

        file.close();
        return 0;
    }

---

## Append to a File

    ofstream file("data.txt", ios::app);

    file << "\nNew Line Added";
    file.close();

---

## File Open Modes

- `ios::in` → read mode  
- `ios::out` → write mode  
- `ios::app` → append mode  
- `ios::binary` → binary mode  

---

## Checking File Open

    ifstream file("data.txt");

    if (!file) {
        cout << "File not found!";
    }

---

## Example: Copy File Content

    #include <fstream>
    #include <iostream>
    using namespace std;

    int main() {
        ifstream src("data.txt");
        ofstream dest("copy.txt");

        string line;

        while (getline(src, line)) {
            dest << line << endl;
        }

        src.close();
        dest.close();
    }

---

## Summary
- Use `<fstream>` for file handling
- `ofstream` writes data
- `ifstream` reads data
- Always close files after use
- Use append mode to add data
