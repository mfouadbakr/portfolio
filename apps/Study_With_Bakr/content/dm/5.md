# 5. Core Mining Techniques

Data mining techniques are used to discover patterns, relationships, and useful information from large datasets.

---

## 1. Classification

Classification is used to assign labels to data.

Example:
- Email → Spam / Not Spam
- Student → Pass / Fail

### Example algorithms:
- Decision Trees
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM)

---

## Python Example (Simple Classification Idea)

    from sklearn.tree import DecisionTreeClassifier

    X = [[1, 0], [0, 1], [1, 1]]
    y = [0, 1, 1]

    model = DecisionTreeClassifier()
    model.fit(X, y)

    print(model.predict([[1, 0]]))

---

## 2. Clustering

Clustering groups similar data without labels.

Example:
- Customer segmentation
- Grouping similar documents

### Algorithm:
- K-Means

---

## Python Example (K-Means)

    from sklearn.cluster import KMeans
    import numpy as np

    data = np.array([[1, 2], [1, 4], [5, 7], [6, 8]])

    model = KMeans(n_clusters=2)
    model.fit(data)

    print(model.labels_)

---

## 3. Association Rules

Find relationships between items.

Example:
- Market Basket Analysis
- "If bread → butter"

### Algorithm:
- Apriori Algorithm

---

## 4. Regression

Used for predicting continuous values.

Example:
- Predicting house prices
- Salary prediction

### Types:
- Linear Regression
- Multiple Regression

---

## Python Example (Regression)

    from sklearn.linear_model import LinearRegression
    import numpy as np

    X = np.array([[1], [2], [3], [4]])
    y = np.array([2, 4, 6, 8])

    model = LinearRegression()
    model.fit(X, y)

    print(model.predict([[5]]))

---

## 5. Anomaly Detection

Detect unusual data points.

Example:
- Fraud detection
- System failures

---

## Summary
- Classification → labeled data
- Clustering → grouped data
- Association → relationships
- Regression → prediction
- Anomaly detection → outliers
