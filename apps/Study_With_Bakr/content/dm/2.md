# 2. NumPy & Pandas

NumPy and Pandas are core libraries for data analysis and manipulation in Python.

---

# NUMPY

## What is NumPy?

NumPy is used for working with arrays and performing fast mathematical operations.

---

## Creating Arrays

    import numpy as np

    arr = np.array([1, 2, 3, 4, 5])
    print(arr)

---

## 2D Arrays

    matrix = np.array([
        [1, 2, 3],
        [4, 5, 6]
    ])

    print(matrix)

---

## Array Operations

    arr = np.array([1, 2, 3])

    print(arr + 10)
    print(arr * 2)
    print(arr ** 2)

---

## Useful Functions

    arr = np.array([1, 2, 3, 4, 5])

    print(np.sum(arr))
    print(np.mean(arr))
    print(np.max(arr))
    print(np.min(arr))

---

## Indexing & Slicing

    arr = np.array([10, 20, 30, 40, 50])

    print(arr[0])     # 10
    print(arr[1:4])   # [20 30 40]

---

## Reshaping Arrays

    arr = np.array([1, 2, 3, 4, 5, 6])

    new_arr = arr.reshape(2, 3)
    print(new_arr)

---

# PANDAS

## What is Pandas?

Pandas is used for structured data (tables, CSV files, Excel).

---

## Creating a DataFrame

    import pandas as pd

    data = {
        "Name": ["Ali", "Sara", "Omar"],
        "Age": [20, 22, 21]
    }

    df = pd.DataFrame(data)
    print(df)

---

## Reading CSV Files

    df = pd.read_csv("data.csv")
    print(df)

---

## Viewing Data

    df.head()     # first 5 rows
    df.tail()     # last 5 rows

---

## Selecting Columns

    print(df["Name"])

---

## Filtering Data

    df[df["Age"] > 20]

---

## Adding Columns

    df["Score"] = [90, 85, 88]

---

## Basic Statistics

    df.describe()

---

## Handling Missing Values

    df.dropna()        # remove missing values
    df.fillna(0)       # replace missing values

---

## Summary
- NumPy = fast numerical arrays
- Pandas = structured data handling
- Both are essential for data mining
