# 6. NLP & Sentiment Analysis

Natural Language Processing (NLP) is used to analyze and understand human language.

Sentiment analysis is a part of NLP that detects emotions in text.

---

## What is NLP?

NLP allows machines to:
- Understand text
- Process language
- Extract meaning

---

## Text Preprocessing

Before analysis, text is cleaned:

- Lowercasing
- Removing punctuation
- Removing stopwords
- Tokenization

---

## Tokenization

Splitting text into words.

    from nltk.tokenize import word_tokenize

    text = "Data mining is powerful"
    tokens = word_tokenize(text)

    print(tokens)

---

## Stopwords Removal

Common words like "is", "the", "and" are removed.

    from nltk.corpus import stopwords

    stop_words = set(stopwords.words('english'))

---

## Stemming

Reduces words to root form.

- running → run  
- playing → play  

---

## Lemmatization

More advanced stemming using grammar rules.

- better → good  
- studies → study  

---

## Sentiment Analysis

Determines whether text is:
- Positive
- Negative
- Neutral

---

## Example using TextBlob

    from textblob import TextBlob

    text = "I love data mining"
    analysis = TextBlob(text)

    print(analysis.sentiment)

---

## Output Meaning

- Polarity → -1 (negative) to +1 (positive)
- Subjectivity → 0 (objective) to 1 (opinion-based)

---

## Bag of Words Model

Converts text into numerical format.

Example:

    "I love AI"
    → [1, 1, 1]

---

## TF-IDF

:contentReference[oaicite:0]{index=0}
