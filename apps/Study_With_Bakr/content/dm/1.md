# 1. Python for Analytics

Python is one of the most popular languages for data analysis, machine learning, and data mining due to its simplicity and powerful libraries.

---

## Why Python for Data Mining?

- Easy to learn and read
- Large ecosystem of libraries
- Strong community support
- Fast prototyping

---

## Basic Python Syntax

    # Printing output
    print("Hello Data Mining")

    # Variables
    x = 10
    name = "Analytics"

---

## Data Types in Python

- int → 10
- float → 10.5
- string → "data"
- bool → True / False
- list → [1, 2, 3]

---

## Lists (Very Important in Data Mining)

    data = [10, 20, 30, 40]

    print(data[0])  # 10
    print(data[-1]) # 40

---

## Looping Through Data

    for value in data:
        print(value)

---

## Conditional Statements

    x = 10

    if x > 5:
        print("Greater than 5")
    else:
        print("Less or equal to 5")

---

## Functions in Python

    def square(x):
        return x * x

    print(square(5))

---

## Introduction to NumPy

NumPy is used for numerical computing.

    import numpy as np

    arr = np.array([1, 2, 3, 4])
    print(arr)

---

## Introduction to Pandas

Pandas is used for data manipulation.

    import pandas as pd

    data = [10, 20, 30]
    df = pd.DataFrame(data, columns=["Values"])

    print(df)

---

## Simple Data Analysis Example

    import numpy as np

    data = [10, 20, 30, 40, 50]

    print(np.mean(data))  # average
    print(np.sum(data))   # sum

---

## Summary
- Python is the base for data mining
- Lists and loops are essential
- NumPy = numerical operations
- Pandas = data handling
