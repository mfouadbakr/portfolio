# 4. Similarity & Distance

Similarity and distance measures are used to compare data points in data mining, clustering, and machine learning.

---

## Why Similarity Matters

We use similarity to:
- Group similar data (clustering)
- Recommend items
- Detect patterns

---

## Euclidean Distance

Measures straight-line distance between two points.


::contentReference[oaicite:0]{index=0}


---

### Example

Points:
- A(1, 2)
- B(4, 6)

Distance:

    d = sqrt((4-1)^2 + (6-2)^2)
    d = sqrt(9 + 16)
    d = 5

## Python Example (Euclidean Distance)

    import numpy as np

    point1 = np.array([1, 2])
    point2 = np.array([4, 6])

    distance = np.sqrt(np.sum((point2 - point1) ** 2))
    print(distance)

---

## Applications

- Recommendation systems (Netflix, YouTube)
- Image recognition
- Clustering algorithms (K-Means)
- Search engines

---

## Summary
- Euclidean = straight line distance
- Manhattan = grid distance
- Cosine = angle similarity
- Jaccard = set similarity
