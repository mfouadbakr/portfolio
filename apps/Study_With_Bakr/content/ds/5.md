# 5. Trees & Binary Search Trees (BST)

A tree is a hierarchical data structure used to represent relationships between nodes.

---

# BASIC TREE TERMINOLOGY

- Root → top node  
- Child → node below another node  
- Parent → node above a child  
- Leaf → node with no children  
- Height → longest path from root to leaf  

---

# BINARY TREE

A binary tree is a tree where each node has at most 2 children.

---

## Node Structure

    struct Node {
        int data;
        Node* left;
        Node* right;
    };

---

## Creating a Node

    Node* createNode(int value) {
        Node* newNode = new Node();
        newNode->data = value;
        newNode->left = NULL;
        newNode->right = NULL;
        return newNode;
    }

---

## TREE TRAVERSALS

---

### 1. Inorder Traversal (Left → Root → Right)

    void inorder(Node* root) {
        if (root != NULL) {
            inorder(root->left);
            cout << root->data << " ";
            inorder(root->right);
        }
    }

---

### 2. Preorder Traversal (Root → Left → Right)

    void preorder(Node* root) {
        if (root != NULL) {
            cout << root->data << " ";
            preorder(root->left);
            preorder(root->right);
        }
    }

---

### 3. Postorder Traversal (Left → Right → Root)

    void postorder(Node* root) {
        if (root != NULL) {
            postorder(root->left);
            postorder(root->right);
            cout << root->data << " ";
        }
    }

---

# BINARY SEARCH TREE (BST)

BST is a special binary tree where:
- Left subtree < root
- Right subtree > root

---

## Insert in BST

    Node* insert(Node* root, int value) {
        if (root == NULL)
            return createNode(value);

        if (value < root->data)
            root->left = insert(root->left, value);
        else
            root->right = insert(root->right, value);

        return root;
    }

---

## Search in BST

    bool search(Node* root, int key) {
        if (root == NULL)
            return false;

        if (root->data == key)
            return true;

        if (key < root->data)
            return search(root->left, key);

        return search(root->right, key);
    }

---

# ADVANTAGES OF TREES

- Fast searching (BST)
- Hierarchical representation
- Efficient sorting and storage

---

# DISADVANTAGES

- Complex implementation
- Requires pointers
- Can become unbalanced

---

# APPLICATIONS

- File systems
- Databases
- Search engines
- AI decision trees

---

# SUMMARY

- Tree = hierarchical structure  
- Binary Tree = max 2 children  
- BST = ordered binary tree  
- Traversals = inorder, preorder, postorder
