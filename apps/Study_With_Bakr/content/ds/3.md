# 3. Linked Lists

A linked list is a linear data structure where elements (nodes) are connected using pointers.

Unlike arrays, linked lists are dynamic in size.

---

## Node Structure

Each node contains:
- Data
- Pointer to next node

    struct Node {
        int data;
        Node* next;
    };

---

## Types of Linked Lists

- Singly Linked List
- Doubly Linked List
- Circular Linked List

---

# SINGLY LINKED LIST

## Creating a Node

    Node* head = NULL;

---

## Inserting at Beginning

    Node* newNode = new Node();
    newNode->data = 10;
    newNode->next = head;
    head = newNode;

---

## Inserting at End

    Node* newNode = new Node();
    newNode->data = 20;
    newNode->next = NULL;

    if (head == NULL) {
        head = newNode;
    } else {
        Node* temp = head;
        while (temp->next != NULL) {
            temp = temp->next;
        }
        temp->next = newNode;
    }

---

## Traversing a Linked List

    Node* temp = head;

    while (temp != NULL) {
        cout << temp->data << " ";
        temp = temp->next;
    }

---

## Deleting a Node (Beginning)

    Node* temp = head;
    head = head->next;
    delete temp;

---

# DOUBLY LINKED LIST

Each node has:
- Data
- Next pointer
- Previous pointer

    struct Node {
        int data;
        Node* next;
        Node* prev;
    };

---

## Circular Linked List

Last node points back to the head.

---

## Advantages of Linked List

- Dynamic size
- Efficient insert/delete
- No memory wastage

---

## Disadvantages

- No random access
- Extra memory for pointers
- Slower traversal

---

## Array vs Linked List

| Feature | Array | Linked List |
|--------|------|-------------|
| Size | Fixed | Dynamic |
| Memory | Contiguous | Scattered |
| Access | Fast | Slow |

---

## Summary
- Linked list uses nodes and pointers
- Dynamic structure
- Efficient insert/delete operations
