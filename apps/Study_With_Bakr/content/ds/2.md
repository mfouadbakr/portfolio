# 2. Arrays & Vectors

Arrays and vectors are used to store multiple values in C++.

---

## Arrays

An array stores elements of the same type in contiguous memory.

---

### Declaring an Array

    int arr[5] = {1, 2, 3, 4, 5};

---

### Accessing Elements

    cout << arr[0];  // first element
    cout << arr[4];  // last element

---

### Loop Through Array

    for (int i = 0; i < 5; i++) {
        cout << arr[i] << endl;
    }

---

### Modifying Elements

    arr[2] = 10;

---

## 2D Arrays

Used to store matrices.

    int matrix[2][2] = {
        {1, 2},
        {3, 4}
    };

---

### Loop Through 2D Array

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

---

## Limitations of Arrays

- Fixed size
- Cannot grow dynamically
- Hard to manage memory manually

---

# VECTORS

Vectors are dynamic arrays in C++.

---

## Declaring a Vector

    #include <vector>

    vector<int> v = {1, 2, 3};

---

## Adding Elements

    v.push_back(4);
    v.push_back(5);

---

## Accessing Elements

    cout << v[0];

---

## Loop Through Vector

    for (int i = 0; i < v.size(); i++) {
        cout << v[i] << endl;
    }

---

## Range-Based Loop

    for (int x : v) {
        cout << x << endl;
    }

---

## Removing Elements

    v.pop_back();

---

## Vector Size

    cout << v.size();

---

## Advantages of Vectors

- Dynamic size
- Built-in functions
- Easier memory management

---

## Arrays vs Vectors

| Feature | Array | Vector |
|--------|------|--------|
| Size | Fixed | Dynamic |
| Memory | Manual | Automatic |
| Functions | Limited | Rich |

---

## Summary
- Arrays = fixed size
- Vectors = dynamic arrays
- Vectors are safer and easier
