# 4. Stacks & Queues

Stacks and Queues are linear data structures used to store and manage data in specific order rules.

---

# STACK

A stack follows **LIFO (Last In First Out)** principle.

Example: plates stacked on top of each other.

---

## Stack Operations

- push → add element  
- pop → remove element  
- top → get last element  
- isEmpty → check if empty  

---

## Stack Using Array

    #define SIZE 5

    int stack[SIZE];
    int top = -1;

---

## Push Operation

    void push(int value) {
        if (top == SIZE - 1) {
            cout << "Stack Overflow";
        } else {
            top++;
            stack[top] = value;
        }
    }

---

## Pop Operation

    void pop() {
        if (top == -1) {
            cout << "Stack Underflow";
        } else {
            top--;
        }
    }

---

## Display Stack

    void display() {
        for (int i = top; i >= 0; i--) {
            cout << stack[i] << " ";
        }
    }

---

# QUEUE

A queue follows **FIFO (First In First Out)** principle.

Example: people in a line.

---

## Queue Operations

- enqueue → add element  
- dequeue → remove element  
- front → first element  
- rear → last element  

---

## Queue Using Array

    int queue[5];
    int front = -1, rear = -1;

---

## Enqueue Operation

    void enqueue(int value) {
        if (rear == 4) {
            cout << "Queue Full";
        } else {
            if (front == -1) front = 0;
            rear++;
            queue[rear] = value;
        }
    }

---

## Dequeue Operation

    void dequeue() {
        if (front == -1 || front > rear) {
            cout << "Queue Empty";
        } else {
            front++;
        }
    }

---

## Display Queue

    void display() {
        for (int i = front; i <= rear; i++) {
            cout << queue[i] << " ";
        }
    }

---

# Stack vs Queue

| Feature | Stack | Queue |
|--------|------|-------|
| Order | LIFO | FIFO |
| Insert | Top | Rear |
| Delete | Top | Front |

---

## Applications

### Stack:
- Undo operations
- Function calls
- Expression evaluation

### Queue:
- Task scheduling
- Printing jobs
- BFS in graphs

---

## Summary
- Stack = LIFO
- Queue = FIFO
- Both are fundamental DS
