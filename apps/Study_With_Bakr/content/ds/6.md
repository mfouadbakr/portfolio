# 6. Heaps & Sorting

Heaps are special tree-based data structures used mainly for priority operations and efficient sorting.

---

# HEAP

A heap is a **complete binary tree** that satisfies the heap property.

---

## Types of Heap

### 1. Max Heap
- Parent node ≥ child nodes
- Largest element is at root

### 2. Min Heap
- Parent node ≤ child nodes
- Smallest element is at root

---

## Heap Representation (Array)

A heap is stored in an array:

- Parent index → (i - 1) / 2  
- Left child → 2i + 1  
- Right child → 2i + 2  

---

# MAX HEAP INSERTION

    void heapifyUp(int heap[], int i) {
        int parent = (i - 1) / 2;

        if (i > 0 && heap[i] > heap[parent]) {
            swap(heap[i], heap[parent]);
            heapifyUp(heap, parent);
        }
    }

---

# HEAP INSERT

    void insert(int heap[], int &size, int value) {
        heap[size] = value;
        heapifyUp(heap, size);
        size++;
    }

---

# HEAPIFY DOWN (Deletion)

    void heapifyDown(int heap[], int size, int i) {
        int largest = i;
        int left = 2 * i + 1;
        int right = 2 * i + 2;

        if (left < size && heap[left] > heap[largest])
            largest = left;

        if (right < size && heap[right] > heap[largest])
            largest = right;

        if (largest != i) {
            swap(heap[i], heap[largest]);
            heapifyDown(heap, size, largest);
        }
    }

---

# HEAP SORT

## Steps:
1. Build heap  
2. Swap root with last element  
3. Reduce heap size  
4. Heapify root  

---

## Heap Sort Code

    void heapSort(int heap[], int size) {
        for (int i = size / 2 - 1; i >= 0; i--) {
            heapifyDown(heap, size, i);
        }

        for (int i = size - 1; i > 0; i--) {
            swap(heap[0], heap[i]);
            heapifyDown(heap, i, 0);
        }
    }

# SORTING ALGORITHMS

---

## Bubble Sort

    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                swap(arr[j], arr[j+1]);
            }
        }
    }

---

## Selection Sort

    for (int i = 0; i < n-1; i++) {
        int min = i;
        for (int j = i+1; j < n; j++) {
            if (arr[j] < arr[min])
                min = j;
        }
        swap(arr[i], arr[min]);
    }

---

## Insertion Sort

    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;

        while (j >= 0 && arr[j] > key) {
            arr[j+1] = arr[j];
            j--;
        }

        arr[j+1] = key;
    }

---

# TIME COMPLEXITY

| Algorithm | Complexity |
|----------|------------|
| Bubble Sort | O(n²) |
| Selection Sort | O(n²) |
| Insertion Sort | O(n²) |
| Heap Sort | O(n log n) |

# SUMMARY

- Heap = complete binary tree  
- Max heap = largest at root  
- Heap sort = efficient sorting method  
- Sorting algorithms vary in efficiency
