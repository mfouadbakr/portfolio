# 8. Graphs (BFS & DFS)

A graph is a non-linear data structure made of nodes (vertices) connected by edges.

---

# TYPES OF GRAPHS

- Directed Graph → edges have direction  
- Undirected Graph → edges have no direction  
- Weighted Graph → edges have weights  

---

# GRAPH REPRESENTATION

## 1. Adjacency Matrix

    int graph[5][5];

If there is an edge:
    graph[i][j] = 1;

---

## 2. Adjacency List (Preferred)

    vector<int> adj[5];

---

# ADD EDGE (UNDIRECTED GRAPH)

    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

---

# DEPTH FIRST SEARCH (DFS)

DFS explores as deep as possible before backtracking.

---

## DFS Implementation

    void dfs(int node, vector<bool> &visited) {
        visited[node] = true;
        cout << node << " ";

        for (int neighbor : adj[node]) {
            if (!visited[neighbor]) {
                dfs(neighbor, visited);
            }
        }
    }

---

# BREADTH FIRST SEARCH (BFS)

BFS explores level by level.

---

## BFS Implementation

    void bfs(int start) {
        vector<bool> visited(5, false);
        queue<int> q;

        visited[start] = true;
        q.push(start);

        while (!q.empty()) {
            int node = q.front();
            q.pop();

            cout << node << " ";

            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    q.push(neighbor);
                }
            }
        }
    }

---

# BFS vs DFS

| Feature | BFS | DFS |
|--------|-----|-----|
| Strategy | Level-wise | Depth-wise |
| Data Structure | Queue | Stack/Recursion |
| Memory | Higher | Lower |
| Use Case | Shortest path | Path exploration |

# SUMMARY

- Graph = nodes + edges  
- DFS = deep traversal  
- BFS = level traversal  
- Adjacency list is most efficient
